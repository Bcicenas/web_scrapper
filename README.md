# Web Scrapper
  Simple GUI application for web scrapping 

# Installation and usage
  To install this project:
   - Download source
   - Create python virutalenv
   - Run ``` pip install . ```
   - Run ScrapperApp.py ``` web_scrapper\ScrapperApp.py ```
